### Legado — Otimizador de Anúncios

Esta pasta guarda a versão original do projeto (Express + OpenAI para gerar relatórios de otimização de anúncios do Mercado Livre). Foi preservada aqui para referência e para que a feature possa ser reintegrada como módulo do dashboard Onicanal numa fase futura.

Para rodar a versão antiga isoladamente:

```bash
cd legacy
node server.js
```

Variáveis necessárias: `OPENAI_API_KEY`, `PORT`.
